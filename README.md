A sleek and powerful web application for real-time stock tracking, visualization, and analysis. This dashboard provides users with the tools to monitor market fluctuations, analyze historical performance, and manage their personal watchlist.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js (v18.x or higher recommended)
- npm or Yarn
- A `.env` file with your API key (see `.env.example`).

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/stocks_app.git
    cd stocks_app
    ```

2.  **Install dependencies:**
    Using npm:
    ```sh
    npm install
    ```
    Or using Yarn:
    ```sh
    yarn install
    ```

3.  **Set up environment variables:**
    Create a `.env` file in the root of the project and add your API key:
    ```env
    REACT_APP_API_KEY=your_api_key_here
    ```

4.  **Run the development server:**
    ```sh
    npm start
    ```
    The application will be available at `http://localhost:3000`.
